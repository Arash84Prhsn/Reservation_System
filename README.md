# Financial technologies lab reservation system
This repository holds all the code regarding both the backend and the frontend of the website
for a reservation system of the seats in the financial technologies laboratory of faculty of
mathematical sciences of the Ferdowsi university of Mashhad.

Further details regarding backend API and frontend design documentation can be found in the 
respective directories.

## The reservation system
The reservation system allows users who have signed up on the site view the availabe time slots
for each seat in the lab to make and cancel reservations. They are allowed to make reservations from
3 days ahead until the end of the next week.
